# 资金计划填报系统 - 项目进度文档

**版本**: 1.2
**更新日期**: 2024-02-26
**项目状态**: 开发中

## 一、项目概述

### 1.1 项目目标
资金计划填报系统旨在提供一个完整的资金计划管理平台，支持多机构、多部门的资金预测、实际支付填报和审核流程。

### 1.2 技术栈
- 前端框架: Next.js 14 (App Router)
- UI组件库: shadcn/ui
- 数据库: SQLite (通过 Prisma ORM)
- 开发语言: TypeScript
- 状态管理: React Context + Hooks
- 表单处理: React Hook Form
- 数据验证: Zod
